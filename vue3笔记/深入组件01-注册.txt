注册&使用是两回事，先注册，后使用。
全局注册的所有其他组件可见，局部注册的仅当前组件可见。

# 全局注册
app的component()方法：
app.component('ComponentA', ComponentA)

# 局部注册
父组件的components对象：
export default {
    components: {
      ComponentA,
    }
}

# 组件名
推荐注册为PascalCase格式，使用时 <PascalCase/> 和 <pascal-case/> 都可以。
vue会把前者都翻译为后者，因为dom本身并不允许PascalCase。
