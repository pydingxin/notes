__________________________________________________________________

ascii('a') → 97
__________________________________________________________________
char(97) → 'a'

__________________________________________________________________
查找子串,找不到返0
charindex('a','sas') → 2
SQL中的字符串s下标为[1,len(s)]
__________________________________________________________________
查找模式子串
PATINDEX('%en_ure%', '--ensureXX') → 3
模式子串必须以'%'开头和结尾
__________________________________________________________________

字符串连接
concat('a','b') → 'ab'
等于 'a'+'b'
__________________________________________________________________
用分隔符连接字符串
concat_ws('-','a','b') → 'a-b'

这些字符串函数可直接用于select，如：
select concat_ws('__',包码,管码,证件号) from new;

len(s)

__________________________________________________________________
left('abcdef',3) → 'abc'
取最左边的n个字符串
__________________________________________________________________
right('abcdef',3) → 'def'
取最右边的n个字符串
__________________________________________________________________
lower('AbCd') → 'abcd'
大写都转为小写
__________________________________________________________________
upper('AbCd') → "ABCD"
小写都转为大写
__________________________________________________________________
删除左边空格 ltrim(' a  ') → 'a  '
删除右边空格 rtrim(' a  ') → ' a'
删除两边空格 trim(' a  ') → 'a'
__________________________________________________________________
quotename('aa') → '[aa]'
主要是写代码时给库名表名加外括号
__________________________________________________________________
替换全部指定子串
replace('dingxinding','ding','_') → '_xin_'

逐个替换 
translate(s,sub1,sub2) 把sub1中对应位置的sub2
translate('dingxin','ni','N1')
translate('dingxin','ni','N1') → d1Ngx1N
__________________________________________________________________
replicate('*',4) → '****'
字符串重复

space(3) → '   '
空字符重复
__________________________________________________________________
reverse('1234') → '4321'
字符串逆转
__________________________________________________________________
str(3.14,5,1) → '  3.1'
str(x,n,m)
数字x转字符串，n为字符串总长，自然包括小数点和符号，m为小数位数，多余左侧填空格，
__________________________________________________________________
string_agg(col,sep) as col_name
用group by聚合搜索时使用
__________________________________________________________________
对json添加转义字符
string_escape('{"name":"ding"}','json') → {\"name\":\"ding\"}

__________________________________________________________________

stuff(s1,idx,len,s2)
填充字符串，把s2填充到s1中，位置为idx，从s1中替换长度为len的子串
应该说是区间替换
stuff('dingxin',2,0,'_') → d_ingxin
stuff('dingxin',2,1,'_') → d_ngxin
__________________________________________________________________
取子串 substring(s,idx,len)
substring('dingxin',3,2) → 'ng'
